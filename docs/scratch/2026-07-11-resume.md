## The locked shape (one paragraph)

A `Framework` is the **sole owner of a project type** — Discover, Scaffold (RICH,
per-framework), Build. **One** files/templating mechanism: `framework/scaffold/`. `baseline/`
**dissolves** (templating → `framework/scaffold/`; embedded infra files + pins + routing → into
the `Infra` framework). Top-level `scaffold/` **dissolves** (mechanism → `framework/scaffold/`;
discover → `framework/`; orchestration → `cmd/init`). **`IsInfra()` and any app-vs-infra
branch are ELIMINATED** — Infra.Scaffold just contributes more (its baseline files,
`strategy="latest"` seed, create-repo need). `builder/` renames to `framework/`. Strategy is a
scaffold-time seed the framework contributes, never a runtime method. Registry creds are
defaulted placeholders (committed CUE, operator hand-edits), never prompted; the picker is gone.
