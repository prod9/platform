## What landed (14 commits, `0d9ffa3`..`07dcbd7`, all green, **unpushed**)

The whole streamline is done. One config spine (`project`), three actions on a repo:

- **`ops/` folded into `project`** — the `Ops` delivery model is a sub-model, no separate pkg.
- **`bootstrapper` → `scaffold`**; `bootstrap` + `ops init` merged into one top-level **`init`**
  (alias `scaffold`). App vs infra is detected by **`builder.IsInfra`** — an **`infra` glob**
  on the dir name (`fi-infra`, `bluepages-infra`, `infra-stage9`, …). `discover` dropped.
- **`engine` `discovery` → `runners`** (Dagger's term; de-collides with `builder.Discover`).
- **`latest` release strategy** — non-versioned, constant `latest`; `publish` skips the
  git-tag lookup for it (`Strategy.IsVersioned()`).
- **`platform/infra` builder** — renders `apps/` (in-process CUE + dsl) and packs the tree into
  a `FROM scratch` image; discovered by the infra glob; `BuildUnit.Vars` carries `[ops.vars]`;
  scaffold sets `strategy=latest`. **oras retired** (`gitops.Publish`/`registry.go`/`oras-go`
  gone; `gitops` keeps only `Render`). Infra delivery is the ordinary `publish`. `ops` group
  dissolved; `render` promoted top-level.
- **Final command surface:** `init build configure exec export ls preview publish release
  render vanity`.
- Docs/specs swept to the new surface; **spec-first governance** established
  ([`docs/README.md`](../README.md)); **oras-drop ADR** recorded
  ([`2026-07-05-infra-publishes-as-plain-image-retire-oras.md`](../decisions/2026-07-05-infra-publishes-as-plain-image-retire-oras.md)).
- Smoke re-recorded for the new surface. **Caveat:** `gobasic`/`pnpmbasic` build entries
  re-recorded as 1m timeouts under this host's parallel-run contention; I corrected them to
  `exitcode 0` (verified 20.8s standalone). **Re-verify `./test.sh` on a machine that meets the
  1m gate under load.**
